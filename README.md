## How To Get Started
###C

Open by Xcode 7.1 and build it.

###Java

Open by IntelliJ IDEA 15.

###Node.js

At first, you should install node_modules.

```
npm install
```

Run the test by this.

```
npm run test
```

###Python

Run the test by this.

```
nosetests tests
```


###Ruby

Run the test by this.

```
rake test
```

###Swift


Open by Xcode 7.1 and build it.


###Golang


Run the test by this.

```
go test
```
